Imperium Theatre

Redesigning a website based on research and a given concept.

Intro

Imperium is a theatre based in Leiden and this year marks their sixtieth anniversary. The website needs a redesign to improve the usability and to gain more members.


Challenges

To draw the attention of the user by introducing different sections on an attractive way.


Goal

A redesign of the current website to allow the users to find information easily and to order tickets on an intuitive way.

Roles

UX/UI/IxD designer


Deliverables

User stories, system requirements, sitemap, task analysis, flowchart, paper prototype


Tools

Axure, Adobe’s Illustrator, Photoshop

user stories

The current website needs a redesign. Content didn’t need a change, just the way it’s placed. To go through the website gives me a good insight on the way the theatre represents itself, this creates a certain image. By looking at the given concept, I was able to create user needs.

prioritizing system requirements

Based on the results of a Heuristic Evaluation, I was able to prioritize the system requirements. Prioritizing in the process is crucial, because I need to determine what is most important. The user wants to get involved with the theatre, this leads to different requirements (subscribing to the newsletter, signing up for a membership etc.). To prevent errors, the system offers guidance through the ordering process with form validations.

New sitemap

The user should go through the website by knowing where to click to find the desired page. By using the card sorting method, I was able to restructure the sitemap.

Visualising the user flow

A flow chart provides me to see how the system reacts on different kinds of input.

Styleguide

A styleguide with suiting colors, this guides me through the design- and coding procesa.
