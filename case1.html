Dutch Bliss

Restructuring and designing the page of an online webshop.

Intro

The Store is a company in the U.S. that delivers Dutch products. With more than hundreds of varieties of food, they’re the premier source for Dutch products for over 65 years.


Challenges

With more than hundreds of varieties of products, it’s crucial to order them properly to achieve a friendly online shopping experience.


Goal

Create a structured layout with a friendly navigation between the website pages.

Roles

UX/UI/IxD designer


Deliverables

Heuristic Evaluations, system requirements, flowchart, user stories and visual designs.


Tools

Adobe's Illustrator and Photoshop

user persona

A persona is a tool that helps to remember the target group, the person represents the real users and helps me to focus on the user needs and goals.

user stories

Based on the persona, I was able to compile user stories. It’s important to focus on the goal and the user needs. In this case Fleur represents the user.

prioritizing system requirements

Based on the results of a Heuristic Evaluation, I was able to prioritize the system requirements. Prioritizing in the process is crucial, because I need to determine what is most important. It’s easy to label everything as “urgent” of “high” priority, but beware: if everything becomes urgent, then everything loses its urgency. To deliver most out of the given time, prioritizing requirements is a must.

Visualising the user flow

A flow chart provides me to go through every step in the user flow. This representation illustrates the way the user interacts with the system and it shows me if there are any problems during this given process.
